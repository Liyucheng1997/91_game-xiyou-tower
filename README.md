# 八十一难 · 西游魔塔

一款无需构建步骤的网页版 2.5D 像素魔塔。孙悟空逐层挑战 81 难，获得猪八戒、沙悟净、小白龙和太白金星的有限帮助，最终在灵山面对如来佛祖并救回唐僧。

[在线游玩](https://liyucheng1997.github.io/xiyou-magic-tower/)

## v1.0 特性

- 81 层确定性地图，26 座钥匙藏宝室，支持上下楼返回挑战
- LV1—LV42 左右的持续修为成长、经验条和属性升级
- 第 4、11、17 难设置可绕行远古劫主，鼓励后期回头挑战
- 九位主线 Boss，最终在灵山面对如来佛祖
- 隐藏宝物、分支奇遇、土地庙和有限伙伴援助
- 21 组 Image Gen 双帧角色动画及逐回合战斗场景
- 三色钥匙、蟠桃、升级和楼层过场特效
- 自动存档、桌面键盘与移动端方向键

## 运行

直接打开 `index.html`，或在目录中启动任意静态服务器：

```powershell
python -m http.server 5173
```

然后访问 `http://localhost:5173`。

## 操作

- `WASD` / 方向键：移动
- `E`：打开妖鉴
- `Esc`：菜单
- 移动端：屏幕方向键

游戏会在每一步后写入浏览器 `localStorage` 自动存档。

## 验证

```powershell
node --check game.js
node tests/audit-game.mjs
```

审计脚本会检查全部 81 层的主路线、藏宝室入口、门钥匙颜色、奖励区可达性、可绕行 Boss，以及完整数值成长曲线。
