# 八十一难 · 西游魔塔

一款无需构建步骤的网页版 2.5D 像素魔塔。孙悟空逐层挑战 81 难，获得猪八戒、沙悟净、小白龙和太白金星的有限帮助，最终在灵山面对如来佛祖并救回唐僧。

[在线游玩](https://liyucheng1997.github.io/xiyou-magic-tower/)

## v2.1 特性

- 全新程序化音频引擎（`audio.js`）：九域各有独立 BGM 主题，调式、速度、音色、混响随地域情绪变化，无需任何音频文件
- 封面 / 标题页专属 BGM，进入游戏后随难域无缝切换；进入战斗自动压低背景乐
- 重做战斗音效：金箍棒重击改为「低频砸击体 + 金属铿锵 + 破空脆响」三层叠加，更具气势
- 分层合成音效：破甲、奖励结算、升级、收法宝、紧箍反噬、毫毛替命、Boss 登场与陨落、通关与败亡各有专属声音
- 主线剧情文本化：九域 Boss 及绕行劫主皆有战前 / 战后台词，火云洞后太白金星点破「这塔即心魔」，并依心魔残片数量分出「悟 / 战 / 迷」三种结局（详见 `docs/剧情设计.md`）

- 入口新增可滚动的八十一难目录，九域分章并支持点击预览迷宫、敌情与路线
- 81 个关卡统一采用“五字 · 五字”诗性对句，并与 Boss、奇遇和地域设计对应
- 照妖镜区域升级为法宝栏，永久收藏并展示法宝、叠加层数、属性和技能说明
- 17 种法宝拥有破甲、横扫、增伤、削弱与减伤等战斗技能及触发特效
- 三尖两刃刀、九齿钉耙、锦斓袈裟、白龙鳞盾和 Boss 专属法宝全面加强

- 81 层高密度确定性迷宫，26 座钥匙藏宝室，支持上下楼返回挑战
- 每层 1—3 道关键封印门、7—16 只怪物及可选择的探索回路
- LV1—LV60 左右的持续修为成长、经验条和属性升级
- 第 4、11、17 难设置可绕行远古劫主，鼓励后期回头挑战
- 九位主线 Boss 均有贴身守卫和专属封印法宝，最终在灵山面对如来佛祖
- 战斗中始终显示怪物实时气血与攻防；无照妖镜强攻可能战死并从第一难重来
- 第三难可取得照妖镜；取得前妖鉴仅显示名称，取得后可提前判断胜负与预计损伤
- 照妖镜归入妖鉴功能区，不占法宝栏；奇遇会预览并明确结算更高的永久奖励
- 标题页提供独立 DEBUG 验关模式，可直接传送至任意一难且不覆盖正式存档
- 新增大圣神通栏：筋斗云可返回已走过的关卡，火眼金睛揭示暗格并破甲，三根救命毫毛可败亡替命
- 菩提祖师、观音菩萨和八卦炉余火构成三段传法剧情，配套全新双帧 NPC 与神通图标
- 新法宝“紧箍咒”永久增加攻击，但每场战斗有概率反噬自身；照妖镜无法预测这一天机变数
- 隐藏宝物、分支奇遇、土地庙和有限伙伴援助
- Image Gen 双帧角色、NPC、道具和 Boss 法宝像素素材
- 三色钥匙、蟠桃、升级、战斗和楼层过场特效
- 九域程序化 BGM 与分层合成音效（详见上方 v1.7 特性）
- 自动存档、桌面键盘与移动端方向键

## 运行

直接打开 `index.html`，或在目录中启动任意静态服务器：

```powershell
python -m http.server 5173
```

然后访问 `http://localhost:5173`。

## 操作

- `WASD` / 方向键：移动
- `E`：打开妖鉴
- `Esc`：菜单
- 移动端：屏幕方向键

游戏会在每一步后写入浏览器 `localStorage` 自动存档。

## 验证

```powershell
node --check game.js
node tests/audit-game.mjs
```

审计脚本会检查全部 81 层的主路线、藏宝室入口、门钥匙颜色、奖励区可达性、可绕行 Boss、五字对句、法宝技能，以及完整数值成长曲线。
